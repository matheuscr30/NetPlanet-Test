## Installation

1. Install the requirements:
    ```bash
    cd frontend/
    npm install
    ```
2. Fill the environment variables:
   * Rename .env.example file to .env inside your project root and fill the base url of the server
   ```bash
   REACT_APP_BASE_URL='http://localhost:8000/api'
   ```
        
## Usage

```bash
npm run start
```
